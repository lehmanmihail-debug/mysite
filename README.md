
# EcoTrack - Платформа для отслеживания экологических инициатив
## ИПо9581 Лехман Михаил
## Описание проекта

EcoTrack - это веб-платформа для мониторинга и участия в экологических проектах и инициативах. Проект создан с целью объединения людей, которые заботятся об окружающей среде, и предоставления инструментов для отслеживания экологических проектов.

## Основные возможности

- **Главная страница** - обзор проекта, статистика, последние новости
- **Блог** - статьи и новости об экологических инициативах
- **О проекте** - информация о миссии, команде и истории разработки
- **Контакты** - форма обратной связи и информация для связи

## Технический стек

- HTML5 (семантическая разметка)
- CSS3 (стилизация и адаптивный дизайн)
- JavaScript (интерактивность, localStorage)
- Git (контроль версий)
- GitHub Pages (хостинг)

## Адаптивный дизайн

Проект разработан с учетом адаптивности для следующих устройств:
- Мобильные устройства (ширина до 576px)
- Планшеты (ширина до 768px)
- Десктоп (ширина от 1200px)

## Функциональность

### Общая
- Единый header и footer на всех страницах
- Навигационное меню с активными ссылками
- Адаптивный дизайн
- Темная/светлая тема (переключатель в правом нижнем углу)

### Главная страница
- Hero-секция с описанием проекта
- Статистика проекта
- Последние новости
- Быстрые ссылки на основные разделы

### Страница "О проекте"
- Описание проекта и его миссии
- Цели и задачи
- Информация о команде
- История разработки в виде таймлайна

### Страница "Контакты"
- Контактная информация
- Форма обратной связи с валидацией
- Часто задаваемые вопросы (аккордеон)
- Карта расположения

### Страница "Блог"
- Список статей с датами публикации
- Боковая панель с поиском, категориями и популярными статьями
- Кнопка "Загрузить больше статей"
